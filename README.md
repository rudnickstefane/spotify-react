# Spotify React (Imersão Dev Alura)

Este projeto é uma imersão em React desenvolvida durante um curso oferecido pela Alura, inspirado na plataforma de música Spotify. O objetivo é criar uma aplicação web que replica algumas funcionalidades e o estilo visual do Spotify, utilizando React e outras tecnologias modernas de desenvolvimento web.

## Funcionalidades

- **Navegação entre páginas**: Explore diferentes seções da aplicação, como playlists, artistas, álbuns, etc.
- **Reprodução de música**: Ouça prévias de músicas e gerencie a reprodução.
- **Interface Inspirada no Spotify**: Uma experiência de usuário similar à do Spotify, com um design atraente e responsivo.

## Tecnologias Utilizadas

- React
- APIs (para integração com serviços de música)
- HTML5
- CSS3
- JavaScript

## Executando o Projeto

Para executar o projeto localmente, siga estas etapas:

1. Clone o repositório para a sua máquina local:

### `git clone https://github.com/rudnickstefane/spotify-react.git`

2. Navegue até a pasta do projeto:

### `cd spotify-react`

3. Instale as dependências:

### `npm install`

4. Inicie a aplicação:

### `npm start`

Isso iniciará o servidor de desenvolvimento e abrirá a aplicação no seu navegador padrão.
